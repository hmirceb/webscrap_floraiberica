# Webscrapping Flora Ibérica

A small project to scrape the web of [Flora Iberica](http://www.floraiberica.es/) for synonyms.

Script 1 parses the HTML code of each genus available in the web. Script 2 formats the names into a usable data frame. The "utils" script contains several auxiliary functions.

All data are accesible under license [CC BY-NC-SA 2.5 ES](https://creativecommons.org/licenses/by-nc-sa/2.5/es/)

The proper citation for this dataset and Flora Iberica as a whole is:

Castroviejo, S. (coord. gen.). 1986-2012. Flora iberica 1-8, 10-15, 17-18, 21. Real Jardín Botánico, CSIC, Madrid.
